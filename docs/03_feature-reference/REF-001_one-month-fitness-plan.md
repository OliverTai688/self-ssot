# 接下來一個月減脂保肌簡表

## 一個月 SMART 目標

在接下來 1 個月內，透過每週 3 次重訓、2 次輕有氧與每日 110–130g 蛋白質攝取，將體重從 72.9kg 降至 71.0–71.5kg、體脂率從 20.8% 降至 19–20% 內，並維持肌肉量 54kg 以上。

---

## 每日飲食原則

### 每日目標

| 項目 | 目標 |
|---|---|
| 熱量 | 約 1800–2100 kcal |
| 蛋白質 | 110–130g |
| 水分 | 2000–2500 ml |
| 步數 | 7000–9000 步 |
| 睡眠 | 7 小時以上 |
| 飲食重點 | 減脂但不餓爆、不掉肌肉 |

### 每餐比例

每餐抓這個比例即可：

```text
蛋白質 1.5–2 掌心
蔬菜 2 拳頭
澱粉 0.5–1 拳頭
油脂少量
```

### 避免項目

- 含糖飲料
- 油炸宵夜
- 晚餐同時高油＋高澱粉
- 連續多天低於 1700 kcal
- 只吃澱粉、蛋白質不足

---

## 每日飲食菜單

> 可照 7 天循環吃，連續執行 4 週。  
> 若外食，只要符合「蛋白質足夠、澱粉適量、少油炸」即可替換。

### Day 1

| 餐次 | 內容 |
|---|---|
| 早餐 | 無糖豆漿＋水煮蛋 2 顆＋香蕉 1 根 |
| 午餐 | 雞胸/雞腿便當，飯半碗，青菜加量 |
| 點心 | 希臘優格或無糖優酪乳 |
| 晚餐 | 豆腐＋魚/雞肉＋青菜，澱粉半碗 |

### Day 2

| 餐次 | 內容 |
|---|---|
| 早餐 | 燕麥＋無糖豆漿＋蛋 1–2 顆 |
| 午餐 | 牛肉/豬里肌飯，飯半碗，避免炸物 |
| 點心 | 茶葉蛋 2 顆或高蛋白飲 |
| 晚餐 | 雞胸沙拉＋地瓜半條 |

### Day 3

| 餐次 | 內容 |
|---|---|
| 早餐 | 全麥吐司 1–2 片＋蛋＋無糖豆漿 |
| 午餐 | 滷雞腿/魚排便當，飯半碗，青菜加量 |
| 點心 | 無糖優酪乳或豆漿 |
| 晚餐 | 火鍋清湯：肉片＋豆腐＋青菜，少加工火鍋料 |

### Day 4

| 餐次 | 內容 |
|---|---|
| 早餐 | 飯糰半顆或御飯糰＋無糖豆漿＋茶葉蛋 |
| 午餐 | 雞肉/魚肉便當，飯半碗，少醬汁 |
| 點心 | 香蕉或水果 1 份＋蛋白質飲品 |
| 晚餐 | 蛋 2 顆＋豆腐＋青菜＋少量澱粉 |

### Day 5

| 餐次 | 內容 |
|---|---|
| 早餐 | 優格＋燕麥＋蛋 1 顆 |
| 午餐 | 牛肉麵可吃，但麵減量，補青菜與蛋白質 |
| 點心 | 茶葉蛋 2 顆 |
| 晚餐 | 雞胸/魚＋青菜＋地瓜半條 |

### Day 6

| 餐次 | 內容 |
|---|---|
| 早餐 | 無糖豆漿＋蛋餅，少醬 |
| 午餐 | 正常外食，但避免炸物與含糖飲 |
| 點心 | 無糖優酪乳或高蛋白飲 |
| 晚餐 | 清淡高蛋白餐：肉/魚/豆腐＋青菜 |

### Day 7

| 餐次 | 內容 |
|---|---|
| 早餐 | 蛋 2 顆＋水果 1 份＋無糖豆漿 |
| 午餐 | 可安排彈性餐，但不要吃到過飽 |
| 點心 | 視飢餓程度補蛋白質 |
| 晚餐 | 回到清淡：蛋白質＋青菜，澱粉少量 |

## 每週運動菜單

### 每週安排

| 星期 | 訓練 |
|---|---|
| 一 | 全身重訓 A |
| 二 | 輕有氧 25–35 分鐘 |
| 三 | 全身重訓 B |
| 四 | 休息或散步 |
| 五 | 全身重訓 A/B 輪替 |
| 六 | 輕有氧 30 分鐘＋伸展 |
| 日 | 休息 |

---

## 全身重訓 A

| 動作 | 組數 |
|---|---|
| 啞鈴高腳杯深蹲 | 3 組 × 8–12 下 |
| 啞鈴臥推 | 3 組 × 8–12 下 |
| 單臂啞鈴划船 | 3 組 × 10–12 下 |
| 啞鈴羅馬尼亞硬舉 | 3 組 × 8–12 下 |
| 平板撐 | 3 組 × 30–45 秒 |

---

## 全身重訓 B

| 動作 | 組數 |
|---|---|
| 保加利亞分腿蹲或弓箭步 | 3 組 × 8–10 下 |
| 啞鈴肩推 | 3 組 × 8–12 下 |
| 伏地挺身或上斜啞鈴臥推 | 3 組 × 8–12 下 |
| 啞鈴臀推或臀橋 | 3 組 × 10–15 下 |
| 側棒式 | 3 組 × 20–40 秒 |

## 有氧與伸展

### 輕有氧

選一種即可：

- 快走
- 慢跑
- 腳踏車
- 橢圓機
- 爬坡走

強度標準：

```text
可以講話，但會微喘。
```

### 久坐伸展

每天選 2–3 個即可：

- 肩頸放鬆 1 分鐘
- 胸椎旋轉 10 次
- 髖屈肌伸展 30 秒 × 2
- 小腿伸展 30 秒 × 2
- 深呼吸收腹放鬆 1–2 分鐘

## 四週執行重點

| 週次 | 重點 |
|---|---|
| 第 1 週 | 建立節奏，不追求操爆 |
| 第 2 週 | 每個動作多 1–2 下或重量微增 |
| 第 3 週 | 飲食最穩的一週，步數拉到 8000–10000 |
| 第 4 週 | 不暴衝，穩定完成並檢查成果 |

## 判斷是否成功

一個月後只要達成以下多數項目，就是成功：

- 體重下降 1.5–2kg
- 體脂率下降約 1–1.5%
- 腰圍下降 2–4cm
- 肌肉量維持 54kg 以上
- 啞鈴重量或次數有進步
- 白天精神與專注力更穩

---

## 核心提醒

這個月不是追求極速變瘦，而是讓身體變得更緊、更穩、更有力量。
