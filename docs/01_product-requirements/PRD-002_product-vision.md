# Product Vision

Personal OS is a personal structure and career competency system for reducing cross-role cognitive load.

It helps the user manage work projects, research directions, relationship context, company strategy, finance drafts, life maintenance, and memories in one coherent system.

## Core Vision

The product is not a generic productivity app. It is a personal context system that helps the user:

- Capture scattered work, research, relationship, company, finance, and life data.
- Convert messy inputs into traceable project memory.
- Separate internal thinking from external/client-facing output.
- Preserve research sources, ideas, and citation context.
- Maintain privacy-first personal and life context.
- Use AI agents as governed collaborators rather than uncontrolled autonomous writers.

## Product Shape

The first operational version should feel like a DB-backed work memory system with a clear path toward the broader Personal OS:

- Work is operational and persisted.
- Client Portal is safe and visibility-filtered.
- PRDs, architecture docs, AGENTS.md, skills, and task backlog form a closed-loop development system.
- Agent Team OS exists as repo governance before runtime agent features.
